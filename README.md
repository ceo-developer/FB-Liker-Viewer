## 🖥️ Installation & Setup (Windows)  
1️⃣ Install Python: [Download Here](https://www.python.org/downloads/)  
2️⃣ Install required modules:  
   ```sh
   pip install -r requirements.txt
   ```  
3️⃣ Run the tool:  
   ```sh
   python facebook_tool.py
   ```
